# Mini Task Manager

A lightweight web application for managing your daily tasks.

## Features

- **Add Tasks:** Quickly add new tasks.
- **Edit Tasks:** Update tasks using a modal popup.
- **Delete Tasks:** Remove tasks with a confirmation.
